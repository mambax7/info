Help: DH-Info

Description

DH-info is a content module to any text, Iframs or even the contents of callend php scripts.

Install/uninstall this moduls

No special measures necessary, follow the standard installation process.
Extract the /profile folder into the ../modules directory. Install the module through Admin -> System Module -> Modules.
This module is clonbar, that is, the module can be used several times. To do this, "only" the Modul name (info) and 2 files must be renamed.
More details in the readme.txt info/doc file

Detailed instructions for the installation of the module are in English as XOOPS Operations Manual available.
